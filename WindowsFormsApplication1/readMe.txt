This project is to demonstrate the use of PL/SQL of Oracle 10G XE. All the pl/sql codes are in project_PLSQL.sql file.

You will need:
1. Oracle 10g XE installed.
2. Change the user name and password of your oracle database in DataBaseConnection Class. 
3. Create all the tables from the given create_table.sql file.
4. Insert all the necessary data from insert_into_table.sql file.
Hint: to load a .sql file, go to cmd. Type sqlplus. Give your user name and password. then @[full path of your .sql file].
5. After all the create tables and insert into table are done. Load the project's .sln file. 
6. Make sure you start your database!
